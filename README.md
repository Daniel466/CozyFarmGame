# 🌾 Cozy Farm Game

A 3D cozy farming game built in Unity 6 with URP, inspired by Farm Together.

## About
A warm, low-pressure 3D farming game where you plant and harvest crops, build and decorate your dream farm, and progress at your own pace. No fail states. No timers. Just cozy. 🌻

## Getting Started
1. Install Unity 6 via Unity Hub
2. Clone this repository
3. Open the project folder in Unity 6 (ensure URP is selected)
4. Hit Play and start farming!
